3 ways to declare a function

1.function declaration

syntax: 

            function function_name(parameters){
                //code
            }
        
        
            //calling  a function
            function_name();

2.function expression 

            const function_name = function(parameters){
                //code
            }
            //call
        function_name(arg)

3. arrow function 

        const function_name=(parameters) =>{
            //code
        }
        //call function_name(arg);

1.functions accepting a parameter and not returning a value

2.functions accepting a parameter and returning a value

3.functions not accepting a parameter and not returning a value
